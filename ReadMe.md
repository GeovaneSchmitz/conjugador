> Um conjugador de verbos da língua portuguesa.

**Instalação**: `npm i -s conjugador` ou `bower i -s conjugador`.

```js
conjugar('amar'); // {eu: 'amo', tu: 'amas', ele: 'ama', ...}
conjugar('pôr'); // {eu: 'ponho', tu: 'pões', ele: 'põe', ...}
```
